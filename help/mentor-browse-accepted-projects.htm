Accepted projects you are mentoring

Once a student has accepted one of your project offers, he/she is recorded in the system as now taking that project during the Semester of code. You should be notified by email that a student has accepted your offer. To access your accepted projects use the 'My accepted projects' link.

You should then see a list of your students.

Click the info icon for a particular student to open the resources for that students project.

Here there are two options.

The 'project agreement' link is a simple form designed to allow you, the student and the supervisor to specify any additional information regarding the project work to be undertaken, such as milestones and delivery dates. (Note - this is optional and only needed if required.)

The 'Project details' link opens a tabbed view of the project within a dialog. The dialog window contains six tabs.

  • Overview - contains an overview of the proposal and allows you to add comments to it, which in turn are emailed back to the student and supervisor.
  • Student Details - lists the students details, which institution he/she is attending and their supervisors details.
  • Solution Summary - the students brief proposal synopsis
  • Solution Description - the students detailed proposal text
  • Project - is a reference back to the project idea you initially created
  • Status - is now set to accepted by the student.
